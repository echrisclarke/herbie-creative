# Card-o-Bot sample assets

Card-o-Bot is a web app that AI-generates trading card graphics users can draw on. Ads cover two stories: a real printed deck, and the app on devices in the wild.

- `card-front.png` — primary user-made custom card front (deck hero)
- `fronts/card-front-*.png` — more user-made custom fronts
- `backs/card-back-*.png` — official back colorways (fixed brand template)
- `devices/phone-ui.png` — Card-o-Bot start UI on a phone (screen fidelity ref)
- `devices/laptop-ui.png` — same UI on a laptop
- `devices/tablet-ui.png` — same UI on a tablet
- `website-screenshot.png` — older flat site screenshot (optional / legacy)
- `logo.png` — Card-o-Bot logo

Brief: deck product uses front + backs; app product uses `devices/*` UI refs for in-environment phone / tablet / laptop ads.
